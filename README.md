Product Inventory Cost Calculator

Description
This project is a simple Python program that allows a user to enter information about a product and calculate the total cost based on its price and quantity.

The program requests the product name, price, and quantity from the user. It also validates numeric inputs to ensure that the price and quantity are entered correctly before performing the calculation.

Features
User input for product name
Numeric validation for price and quantity
Automatic calculation of the total cost
