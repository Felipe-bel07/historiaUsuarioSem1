# Product Inventory Cost Calculator

## Description
This project is a simple Python program that allows a user to enter information about a product and calculate the total cost based on its price and quantity.
The program is designed to request the product name, price, and quantity from the user. It also validates numeric inputs to ensure that the price and quantity are entered correctly before performing the calculation.

## How it works
1. The program asks the user to enter the product name.
2. It requests the price of the product.
3. The program asks for the quantity.
4. It multiplies the price by the quantity to calculate the total cost.
5. Finally, the program displays the product information and the total cost.

## Status
The program is currently running as a simple command-line application and it is processing user input to calculate the total cost.
